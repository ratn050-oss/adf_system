## ALUR YANG BENAR:

### TAHAP 1: SETUP USER OWNER (dari Developer Menu)

1. **Login ke Developer Panel**
   URL: http://localhost:8081/adf_system/developer/login.php
   Username: developer
   Password: dev123 (atau sesuai setup Anda)

2. **Buat User Owner**
   - Klik menu "Users" di developer panel
   - Klik tombol "Create New User"
   - Isi form:
     * Username: owner (atau nama lain)
     * Email: owner@example.com
     * Full Name: Owner Name
     * Password: (buat password aman)
     * Role: Pilih "Owner" 
     * Active: Centang
   - Klik "Save"

3. **Assign Business Access ke Owner**
   - Masih di developer panel, klik menu "Business Users"
   - Pilih user owner yang baru dibuat
   - Assign ke bisnis yang ingin dimonitor:
     [ ] Narayana Hotel
     [ ] Bens Cafe
     [ ] Atau bisnis lainnya
   - Save

4. **Logout dari Developer Panel**


### TAHAP 2: LOGIN SEBAGAI OWNER

3. **Isi Credentials:**
   Username: owner (yang dibuat di tahap 1)
   Password: (password yang dibuat di tahap 1)
